# Jewlexx Tap

## How do I install these formulae?

`brew install jewlexx/tap/<formula>`

Or `brew tap jewlexx/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "jewlexx/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
